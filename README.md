# 🚵 Adventure Works — Supplier Performance Analysis
### Power BI | Power Query | Data Transformation

---

## 📌 Project Overview

This project was built to help **Adventure Works** make data-driven decisions as the company expands into **South America 🌎**. The goal was to identify top-performing suppliers across key product lines — mountain bikes, road bikes, and accessories — using **Delivery Timeliness** as the primary evaluation metric.

> *"The visualization is just the final 10%. The other 90% is knowing how to handle messy, real-world data — and having the discipline to structure it right."*

---

## 🎯 Business Objective

Adventure Works needed a clear, data-backed view of which suppliers could reliably support their South American expansion. This required transforming raw, inconsistent supplier data into a unified, analysis-ready dataset before any dashboard could be built.

---

## 🛠️ What Was Done

| Step | Description |
|------|-------------|
| 🧹 **Data Cleaning** | Filtered out incomplete and unreliable supplier records |
| 🔗 **Data Merging** | Standardized and merged datasets across product categories |
| 🔄 **Data Pivoting** | Pivoted supplier data to surface meaningful patterns |
| 📦 **Data Appending** | Combined multiple datasets for a unified analysis |
| 📊 **Supplier Evaluation** | Ranked suppliers by Delivery Timeliness per product line |
